### 以太坊 EVM原理与实现
* https://www.cnblogs.com/helloworld2018/p/8998926.html

以太坊底层通过EVM模块支持合约的执行与调用，调用时根据合约地址获取到代码，生成环境后载入到EVM中运行。通常智能合约的开发流程是用solidlity编写逻辑代码，再通过编译器编译元数据，最后再发布到以太坊上。
