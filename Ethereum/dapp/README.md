# Dapp

## Dapp 开发步骤
1. 项目分析 
2. 以太坊私链或者测试链
3. 创建工程目录
4. 环境配置
5. 编写合约
6. 合约编译
7. 合约部署
8. 控制台交互
9. 网页交互
10. 测试
