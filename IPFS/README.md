# IPFS
IPFS旨在取代HTTP并为我们所有人构建更好的网络。每个文件及其中的所有块都被赋予一个称为加密哈希的唯一指纹。每个网络节点仅存储它感兴趣的内容，以及一些索引信息，帮助确定谁在存储什么。
