# TRON
TRON是一个雄心勃勃的项目，致力于建立真正分散的互联网及其基础设施。 TRON协议是世界上最大的基于区块链的操作系统之一，为TRON生态系统中的所有分散应用提供基本公共区块链支持，具有高吞吐量，高可扩展性和高可用性。
